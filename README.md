# 🏆 Achievement Badges for Jellyfin

A premium progression and achievement system for Jellyfin that rewards users based on real viewing activity.

---

## ✨ Overview

Achievement Badges enhances Jellyfin with:

- 🎯 Unlockable achievements  
- 🔥 Watch streak tracking  
- 🏆 Rarity-based badge system  
- 👤 Profile showcases  
- 📊 Leaderboards and stats  

Designed to integrate cleanly with modern Jellyfin setups and themes like NetFin.

---

## 🧩 Core Features

- 🏅 Large achievement catalogue  
- 🎖 Rarity tiers (Common → Mythic)  
- 📈 Progress tracking  
- 🔥 Streak system  
- 🏠 Dashboard integration  
- 📂 Sidebar integration  
- 👤 Profile badge showcase  
- 🏆 Leaderboards  

---

## ⚙ Installation

1. Go to **Dashboard → Plugins → Repositories**
2. Add:

```
https://raw.githubusercontent.com/ZL154/AchievementBadges_for_Jellyfin/main/manifest.json
```

3. Save and refresh plugins  
4. Install **Achievement Badges**  
5. Restart Jellyfin  

---

## 🔧 Requirements

- Jellyfin **10.11+**
- ✅ **File Transformation plugin (REQUIRED)**

This plugin uses File Transformation to inject:

- Sidebar entry  
- Dashboard UI  
- Profile showcase  
- Achievements page  

Without it, UI features will not appear.
